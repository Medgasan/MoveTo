MoveTo ha sido diseñado para trabajar con Windows 2000, Windows® XP, Windows® Server 2003, Windows Vista™ & Windows 7.

Mínimo

  • Permisos de administrador sobre el sistema
  • Pentium® class CPU - 1Ghz
  • 1024 x 768 screen resolution
  • 256 Mb RAM
  • Windows® 2000, Windows® XP

Recomendado

  • Permisos de administrador sobre el sistema
  • Pentium® Class CPU - 2Ghz or higher
  • 1280 x 1024 screen resolution
  • 1Gb RAM
  • Windows® XP, Windows Server 2003, Windows Vista™, Windows 7
